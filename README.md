# 📝 CRUD Application

A full-stack **CRUD (Create, Read, Update, Delete)** web application built using **Node.js, Express.js, MongoDB, and EJS**. This project demonstrates complete database operations, RESTful routing, and server-side rendering with a clean and user-friendly interface.

## 🚀 Features

* ➕ Create new chats
* 📖 View all chats
* ✏️ Update existing chats
* 🗑️ Delete chats
* 📦 MongoDB database integration
* 🌐 RESTful routing
* 🎨 Dynamic UI with EJS templates
* 📱 Responsive and clean interface

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* HTML5
* CSS3
* JavaScript

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/crud-application.git
```

2. Navigate to the project directory

```bash
cd crud-application
```

3. Install dependencies

```bash
npm install
```

4. Start the MongoDB server.

5. Run the application

```bash
node app.js
```

Or, if you're using Nodemon:

```bash
nodemon app.js
```

6. Open your browser and visit:

```text
http://localhost:8080
```
